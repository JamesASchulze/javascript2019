Plugin 1: Validator

Validator Page

This plugin was designed to help with form validation.

I used this plug in to validate a demo form. It made it quite easy and I really like this plugin!

jqueryvalidation.org

Plugin 2: Nao Calendar

Nao Calendar Page

Nao Calender is a nice looking date picker/calender plugin.

I used it to display the date picked on the screen.

github.com/Naeemur/jquery-nao-calendar

Plugin 3: QR Code Generator

QR Code Generator Page

This plugin takes text or links and converts them to a QR code format.

I used it to convert some text (which was displyed when I used the built in camera app on my iPhone), and several links to websites.

jeromeetienne.github.io/jquery-qrcode

Plugin 4: Adaptive Backgrounds

Adaptive Backgrounds Page

This plugin identifies one of the primary colors in a photo and uses it as the background for the images parent element.

I used it to draw out colors from some puppy images and make it the background color of the div container of each respective image.

briangonzalez.github.io/jquery.adaptive-backgrounds.js

Plugin 5: BgSwitcher

BgSwitcher Page

This plugin takes in an array of images to cycle through and auto-magically switch backgrounds.

I used it to cycle through the puppy photos used in several of the previous pages.

rewish.github.io/jquery-bgswitcher
