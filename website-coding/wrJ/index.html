Website Review

This website review is on lego.com.

This looks like a lego/starwars website should. You've got the childish, yet fun lego antics with the well known and loved starwars characters. I enjoyed clicking around and watching the videos. There are so many. It would be really easy to sit and watch all of the videos on here. I think they did a good job at mingling the brands together.

I think it was a wise decision for the developers to use the video tag instead of flash. One reason is, flash is dying out. Not to mention, it is notorious for security risks. Developers just don't need flash anymore, there are other options out there now.
