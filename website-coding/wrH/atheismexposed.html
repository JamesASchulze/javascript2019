Website Review

This website review is on atheismexposed.com. When I found this site, it was like a wormhole to the past. I saw colors and page layouts which I had not seen in many moons. It seemed I was back in the 90's! I began to look at the HTML and it was at that point in which I realized why CSS is so convenient. This "tbody" had this "tr", which had this "td", which had this "p", which had a "span" and a "b", with a "span" of it own; not to mention the text color, background color, font size, spacing and all the other stylistic data to make a web page somewhat esthetically pleasing. People are very creative and can do some amazing things with strict parameters. It seems to me that HTML5 and CSS allow more options and flexibility as well as making it easier to have consistency through out the site.
